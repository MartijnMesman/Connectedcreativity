# Audio Files Directory

This folder contains audio files for the Connected Creativity course.

## How to add audio files:

1. **Place your audio files here** (in the `public/audio/` folder):
   - `mindwandering-meditation.mp3`
   - `walking-meditation.mp3` 
   - `grounding-meditation.mp3`
   - etc.

2. **Supported formats:**
   - MP3 (recommended)
   - WAV
   - AAC
   - OGG

3. **File naming:**
   - Use lowercase
   - Use hyphens instead of spaces
   - Be descriptive

## Example files you might add:
```
public/audio/
├── mindwandering-meditation.mp3
├── walking-meditation.mp3
├── grounding-meditation.mp3
├── creative-flow-music.mp3
└── breathing-exercise.mp3
```

## Testing:
After adding files, you can test them by visiting:
`http://localhost:3000/audio/your-filename.mp3`
